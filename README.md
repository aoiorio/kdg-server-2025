### ✋🏼 Commands for the future

- イメージに名前をつけてbuildができる
```
docker build . -t kdg-nginx
```

- imageの一覧を確認する
```
docker image ls
```

- 指定したimageを実行
```
docker run -it kdg-nginx
```
